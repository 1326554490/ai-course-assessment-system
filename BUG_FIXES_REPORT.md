# AI 课程评测系统 - Bug 修复完成

## 已修复的问题

### ✅ 教师端备课预览学生端按钮优化

**问题描述：**
从教师端备课页面（CourseEditor）点击"预览学生端"时，如果课程还没有添加课节，会跳转到无效的URL，导致404错误。

**根本原因：**
- 预览链接使用 `currentLessonId` 作为URL参数
- 如果课程刚创建还没有课节，`currentLessonId` 是空字符串
- 导致链接变成 `/student/course/${courseId}/lesson/`（结尾缺少lessonId）

**修复内容：**
在 `src/pages/teacher/CourseEditor.tsx` 中添加了条件判断：
- **有课节时**：显示可点击的预览按钮（Link组件）
- **无课节时**：显示禁用状态的按钮，提示"请先添加课节"

**修改文件：**
- `src/pages/teacher/CourseEditor.tsx` (第577-594行)
